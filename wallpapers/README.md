Put wallpaper images here for `hyprpaper`.

Supported formats:
- `.jpg`
- `.jpeg`
- `.png`
- `.webp`

`launch-hyprpaper.sh` uses the first image in this directory alphabetically and applies it to every active monitor.

If you want the lock screen to match, also update the `path = ...` entry in `~/.config/hypr-material3/hypr/hyprlock.conf`.
